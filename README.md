# Study Performance Simulator

This project simulates student study habits, generates performance data, analyzes patterns, and visualizes insights using Python, NumPy, Pandas, and Matplotlib.

## Project Idea

The goal is to understand how factors like study hours, sleep, consistency, revision, previous knowledge, and distractions affect exam performance.

## Pipeline

Simulation → Data Generation → CSV Storage → Analysis → Visualization → Insights

## Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Git/GitHub
